  • displayLogs(logsDirectoryName:shareFormat:)

    Displays a full page view with logs that were stored with an use of IteoLoggerStorageItemConsumer.

    Note

    Use the same logsDirectoryName in IteoLoggerStorageItemConsumer to see results. Default shareFormat is: [level] [[date] [time]] - [module_prefix] [module_name]: [output] You can use these parameters in your own format to fill it with data.

    Declaration

    Swift

    func displayLogs(logsDirectoryName: String? = nil, shareFormat: String? = nil)

    Parameters

    logsDirectoryName

    optional directory name where logs are stored.
    shareFormat

    string format which is used when user wants to share logs with someone.
